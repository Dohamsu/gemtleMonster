---
description: 런타임 오류와 린트 오류를 자동으로 식별하고 수정합니다.
---

1. `npm run lint` 명령어를 실행하여 린트 오류를 확인합니다.
2. `npm run build` 명령어를 실행하여 타입 및 빌드 오류를 확인합니다.
3. 발생한 오류 로그를 분석합니다.
4. 각 오류에 대해 원인을 파악하고 코드를 수정합니다.
   - 린트 오류 (예: unused variables)는 해당 코드를 삭제하거나 수정합니다.
   - 타입 오류는 올바른 타입을 정의하거나 타입 단언/가드를 추가합니다.
5. 수정 후 `npm run lint`와 `npm run build`를 다시 실행하여 모든 오류가 해결되었는지 검증합니다.
